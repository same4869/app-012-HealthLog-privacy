隐私政策

生效日期：2026年4月10日

健康手账尊重并保护您的隐私。核心原则：您的所有健康数据均仅保存在您自己的设备本地，应用不设任何服务器，不向任何第三方传输数据。

1. 我们收集与存储哪些数据

数据类型存储位置用途
血压、心率、体重记录设备本地（PersistenceV2）核心功能：记录与趋势分析
备注与状态标签设备本地丰富记录内容
成员档案（姓名、身高、目标值）设备本地个性化展示与健康目标计算
提醒设置（时间、重复日期）设备本地创建系统提醒通知
血压分类标准偏好设备本地统一各页面的分类口径

所有数据存储在 HarmonyOS 应用沙盒内，不会上传至任何服务器，不会同步至云端，不用于广告或营销目的。

2. 应用申请的权限

权限用途是否必须
PUBLISH_AGENT_REMINDER创建本地系统提醒通知（测量血压提醒）否，关闭后仅无法使用提醒功能

应用不申请以下权限：位置、通讯录、相机、麦克风、读取剪贴板、网络访问、广告标识符。

3. 数据导出与导入

您可随时通过「设置 → 导入导出」将数据导出为 CSV 或 JSON 文件，保存到本地或分享给自己。导入功能支持从本地文件或手动粘贴文本恢复数据。导出文件由您完全掌控，应用不保留副本，也不上传到任何位置。

4. 数据共享与第三方

我们不会将您的任何数据出售、出租或共享给任何第三方。应用内不含任何第三方广告 SDK、分析 SDK 或追踪库。

应用仅使用以下 HarmonyOS 系统能力（受华为/系统隐私政策约束）：

  • agent_reminder — 本地提醒通知（可选）
  • PersistenceV2 — 应用沙盒内的本地持久化存储
  • file.picker — 用于导入导出时的文件选择与保存

5. 数据安全

所有数据存储在 HarmonyOS 应用沙盒内，其他应用无法访问。我们没有后端服务器，也不存在数据传输过程，因此不存在服务器端数据泄露风险。

6. 您的数据权利

  • 导出：可通过「设置 → 导入导出」随时导出全部记录为 CSV / JSON 文件；
  • 删除：可在「设置 → 本地存储」中清空当前成员的全部记录；卸载应用将清除设备上的全部数据；
  • 更正：可在应用内随时编辑或删除任意记录。

7. 儿童隐私

本应用不面向 14 岁以下未成年人，我们不会有意收集未成年人的个人信息。

8. 本政策的变更

若本隐私政策有重大变更，我们将在应用更新说明中告知并更新本页面顶部的生效日期。继续使用应用即表示您接受更新后的政策。

Privacy Policy

Effective Date: April 10, 2026

HealthLog respects and protects your privacy. Core principle: all your health data is stored only on your own device — the app has no servers and transmits no data to any third party.

1. Data We Collect and Store

Data TypeStorage LocationPurpose
Blood pressure, heart rate, weight recordsOn-device (PersistenceV2)Core feature: logging and trend analysis
Notes and status tagsOn-deviceEnrich records
Member profiles (name, height, targets)On-devicePersonalized display and health goal calculation
Reminder settings (time, repeat days)On-deviceSchedule system reminder notifications
Blood pressure standard preferenceOn-deviceUnified classification across all views

All data is stored within the HarmonyOS app sandbox. It is never uploaded to any server, synced to the cloud, or used for advertising or marketing.

2. Permissions Requested

PermissionPurposeRequired
PUBLISH_AGENT_REMINDERCreate local system reminder notifications for measurement remindersNo — the app works fully without it; only reminder scheduling is unavailable

The app does not request: location, contacts, camera, microphone, clipboard read access, network access, or advertising identifiers.

3. Data Export and Import

You can export all records at any time as CSV or JSON files via Settings → Import/Export, and save or share them as you wish. Import supports restoring data from local files or pasted text. Exported files are entirely under your control — the app retains no copy and uploads nothing.

4. Data Sharing & Third Parties

We do not sell, rent, or share any of your data with any third party. The app contains no third-party advertising SDKs, analytics SDKs, or tracking libraries.

The app uses only the following HarmonyOS system capabilities (each governed by Huawei/system privacy policies):

  • agent_reminder — local reminder notifications (optional)
  • PersistenceV2 — local persistent storage within the app sandbox
  • file.picker — file selection and saving for import/export

5. Data Security

All data is stored in the HarmonyOS app sandbox, inaccessible to other apps. We operate no backend server and there is no data transmission, eliminating any server-side data breach risk.

6. Your Rights

  • Export: Export all records as CSV / JSON at any time via Settings → Import/Export;
  • Delete: Clear all records for the current member via Settings → Local Storage; uninstalling the app removes all data from your device;
  • Correct: Edit or delete any record inside the app at any time.

7. Children's Privacy

This app is not directed at children under 14, and we do not knowingly collect personal information from minors.

8. Changes to This Policy

If we make material changes to this Privacy Policy, we will note them in the app update release notes and update the effective date at the top of this page. Continued use of the app constitutes acceptance of the updated policy.
